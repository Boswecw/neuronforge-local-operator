# Test Proofread 001

This is a simple test passage for proofreading. It contains a few grammar issues, awkward phrasing, and some inconsistant word usage.

The purpose of this file is to give us a controlled input that we can use repeatedly while comparing model behavior. We want to see whether a model corrects mistakes cleanly, avoids overediting, and preserves intended meaning.

The writer have a strong voice, and that voice should not be flattened without reason. Some models changes too much. Others do not change enough. This sample should help us evaluate that balance.
