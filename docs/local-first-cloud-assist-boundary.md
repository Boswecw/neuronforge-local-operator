# Local-First And Cloud-Assist Boundary

**Status:** source doctrine baseline; cloud assist not fully implemented here.
**Scope:** local-system proving repo.

## Local-First Rule

NeuronForge Local must remain useful without cloud access. Local experiment
records, local model baselines, deterministic validation scripts, and operator
review remain the source proving layer.

## Cloud-Assist Rule

Cloud assist is optional, visible, entitled, receipt-backed, and degraded when it
cannot be used. It must not be a hidden startup requirement or hardcoded fallback
from AuthorForge.

## Current Source Truth

- Existing source service routes are local.
- Local model and prompt registries document proofread/style-analysis baselines.
- GNAT semantic handoff records candidate posture and refuses source-receipt
  mutation.

## Not Yet Source-Proven

The support copy describes cloud escalation and quota-aware routing. This source
repo does not yet prove the full AuthorForge cloud-assist router. Treat that as
planned until source schemas, routes, tests, and receipts exist here.

## Degraded State Rule

Unreachable local runtime, unsupported task, missing entitlement, disabled cloud
route, or incompatible contract must produce an explicit degraded/unavailable
state, never a silent provider call.
